# TFM-Deteccion-Multiclase

El uso de drones y técnicas de Deep Learning se ha consolidado como una herramienta prometedora para la monitorización y conservación de fauna marina. Las capturas de imágenes aéreas y su análisis mediante sistemas de visión por computador permiten obtener información de valor ecológico de forma no invasiva, reduciendo costes y mejorando la eficiencia respecto a métodos tradicionales de monitorización marina. 

En este contexto, el Trabajo Final de Máster estudia la viabilidad de adaptar un sistema de detección originalmente basado en un enfoque monoclase hacia un entorno multiclase capaz de diferenciar distintas especies de megafauna marina, incluyendo tortugas marinas, tiburones y ballenas. Para ello, se construyó un entorno experimental basado en imágenes capturadas mediante drones y se entrenó un modelo YOLOv8 orientado a la detección de distintas especies de megafauna marina. A lo largo del trabajo se realizó un análisis exploratorio del dataset y una evaluación cuantitativa y cualitativa del comportamiento del modelo bajo diferentes condiciones de captura. 

Los resultados obtenidos muestran que el sistema fue capaz de aprender características visuales diferenciables entre especies y mantener un rendimiento consistente incluso en escenarios visualmente complejos. Asimismo, el trabajo permitió identificar limitaciones relevantes asociadas a la variabilidad visual de las imágenes y a la propia estructura del dataset utilizado. 

En conjunto, los resultados sugieren que los modelos de detección basados en Deep Learning presentan un elevado potencial como herramientas de apoyo para automatizar tareas de monitorización y conservación de la biodiversidad mediante drones.
